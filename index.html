Recent Work

Delivery Quadcopter

This project was done through the Harvard-HKUST Summer Research Experience in a group of 8 undergraduates. We built a quadcopter that can perform deliveries to open spaces with the simple use of an app interface to give the delivery address. Once given an address, the latitude and longitude are found. The autopilot then generates waypoints to take it to the destination, carrying the package with a gripper. When it arrives, it lowers the package safely to the ground. It then proceeds to return to its home destination for another delivery.

Sleep Tracking Smartwatch

This is a design for a PCB that can be used for a smartwatch. It combines an accelerometer and a heart rate sensor to track the duration and quality of sleep of the user. It uses an e-ink display to display a digital watch, graphs, and sleep data. For an average amount of use, the watch should be able to work for ~48 hours before needing to be recharged.

Inductive Door Opener

Worked in a group of two to create a device that could unlock a door after having a correct passcode being “knocked” in. This project utilized a magnetic coil as an NFC device. A piezoelectric sensor generated voltages (from knocking on the door) that were sent through the coil after being passed through an AM. A Hall effect sensor picked this up and demodulated the signal before sending it to a microcontroller, which analyzed the code and unlocked the door.

Conway's Game of Life Music Stream

This project (coded in OCaml) constructed Conway’s Game of Life and turned the resulting matrix into a music stream (each row a note, each column a beat). A genetic algorithm was then applied to the music stream in order to try and make the music more enjoyable to listen to.

Medical Tricorder

This project was constructed to bring together all the immediate medical-sensing technologies an EMT might need when they get to an emergency. In a group of three, we built an EKG, hacked a heart-rate monitor/blood-oxygenation sensor, and used a temperature sensor. All of these fed into an Arduino, which would analyze the inputs and print them to a thermal printer.
